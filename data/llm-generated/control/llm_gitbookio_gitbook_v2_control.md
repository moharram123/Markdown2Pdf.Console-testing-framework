## Setup Instructions

Follow these steps to get your environment ready.

- First item: Install Python 3.9 or higher.
- Second item: Clone the repository from the official source.
- Third item: Set up a virtual environment.
- Fourth item: Install all required dependencies.
- Fifth item: Configure environment variables.
- Sixth item: Run the initial database migration.
