To get started with Project Phoenix, follow these steps carefully:

- First item: Install Python 3.8 or above.
- Second item: Clone the repository.
- Third item: Set up your virtual environment.
- Fourth item: Install dependencies.
- Fifth item: Configure environment variables.
- Sixth item: Run initial database migrations.
